# Base de Conhecimento RAJ 10.1

Este diretório conterá documentos e informações específicas da RAJ 10.1:

- Contratos regionais
- Normativas locais
- Procedimentos específicos
- Histórico de notificações
- Base de conhecimento para o Copilot

## Estrutura Futura

```
raj_10_1/
├── contratos/          # Documentos contratuais
├── normativas/         # Normas e procedimentos
├── templates/          # Templates de documentos
└── historico/          # Histórico de ações
```

## Integração com IA

Os documentos desta pasta servirão como base de conhecimento para:
- Respostas contextuais do Copilot
- Geração de notificações
- Orientações aos fiscais
- Recomendações automáticas
